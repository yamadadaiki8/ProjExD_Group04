# ヤマダストライク

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
* 山田くんが画面を縦横無尽に駆け回って、敵を殴って倒す大人気ゲーム

## ゲームの遊び方
* マウス（カーソル）でキャラを引っ張って、指定した方向にはじいて、敵に当てていく
* 全ての敵を倒すことができればゲームクリア

## ゲームの実装
### 共通基本機能
* ゲーム内のUIの実装
* キャラクターの描画
* キャラクターを引っ張って、はじくことのできる機能
* 敵に当たると反射する機能

### 分担追加機能
* ギミック機能（担当：伊藤輝）：ステージ上に3ターン毎にブロックを設置して、障害物の役割をさせるクラス
* Boss機能（担当:原海都）ステージ上にBossの配置。また２ターンに1回十字のレーザーを打ち、当たれば残りのターンを1ターン減る。
* アイテム機能(担当：山田大貴)：画面右下に配置。スペースを押すことで1ターン増加
* コンボ機能(担当:新井拓真)：味方の当たると１体につき１回敵にダメージを与える球を投げる
* SS(ストライクショット)機能(担当:長谷部)：敵を3体撃破すると発動可能.1ターンですべての味方を同時に動かすことができる.
